# Reading Order

Use this order when reconstructing implementation intent from Stitch:

1. `.stitch/DESIGN.md`
2. `.stitch/UI_FLOW.md`
3. approved `design-output/<pass-name>/README.md`
4. `.stitch/designs/*.screen.json`
5. `.stitch/designs/*.html`
6. `.stitch/designs/*.jpeg`
7. any existing app code in the target folder

Do not infer route structure from a single screen in isolation when the full pass is available.
